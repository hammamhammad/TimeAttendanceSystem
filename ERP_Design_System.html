Interaction Behavior Summary

User starts typingAfter 2+ characters, dropdown appears showing matching records (filtered by name, code, or reference number). Results update as user types. Search match is highlighted in results.
Dropdown result rowEach row shows: entity icon + Name (primary, highlighted match) + Reference Number (monospace) + Code (monospace). Hover highlights row.
User selects a recordClick or Enter key. Dropdown closes. Field shows selected record name as a blue link with external-link icon + reference number. Input switches to selected state.
Clicking the linkOpens the selected record's detail page in a new browser tab. URL format: /entity-type/record-id
Clear button Removes the selection, returns field to empty state. Also clears all dependent lookup fields.
Change button Switches back to search mode with the current name pre-filled. User can type over to search for a different record.
Keyboard navigation↑ ↓ navigate dropdown items · Enter selects highlighted item · Esc closes dropdown · Tab moves to next field
